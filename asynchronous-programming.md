# Asynchronous Programming

`async` funciton - Marks function as _asynchronous_
* Returns _promise_
* Needs to use _await_
<br><br>

`await` - Used inside an _async_ function to wait for a _promise_
* try/catch/finally Heavily used with async/await for error handling
<br><br>

`Promise` - Value that will exist later. Interact with _Promises_ using:
* async
* await
<br><br>

`fetch` - Function, makes HTTP requests, returns a _Promise_ that resolves toa Response object.
